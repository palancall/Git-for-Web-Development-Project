Stretch Questions

    1. What is the difference between an inline element and a block element?
        block elements force a new line and inline elements don't. We can't put a block element inside of an inline element, but inline elements can go inside of block elements.
        
    2. What happens when an element is positioned absolutely? 
        it's position is fixed.

    3. How do I make an element take up only the amount of space it needs but also have the ability to give it a width? 
        by putting it in a div.

    4. Name 3 elements that are diplay block by default, 2 elements that are display inline by default and 1 element that is display inline-block by default
        Display block: <p> , <li> , <article>
        Display inline: <a> , <input>
        Display inline-block: <button>

    5. In your own words, explain the box model. What is the fix for the box model? 
        It is a box that wraps around every html element. We use the element and it's marigins, padding and border in order to position the element. Every element in CSS is a box.
